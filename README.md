# 📱 iPhoneCongo

Plateforme de vente et d'achat d'iPhone en République Démocratique du Congo.

## 🇨🇩 Caractéristiques

### Pour les acheteurs
- 📱 Large sélection d'iPhone (neufs et d'occasion)
- 🔍 Recherche avancée par modèle, ville, prix
- 💰 Prix en dollars USD (marché congolais)
- 🤝 Contact direct via WhatsApp
- 📍 Filtrage par ville (Kinshasa, Lubumbashi, Goma, etc.)
- ⚡ Interface rapide et responsive

### Pour les vendeurs (Admin)
- ➕ Publication d'annonces en 2 minutes
- 📝 Formulaire complet avec toutes les spécifications
- 🗑️ Gestion facile des annonces
- 📊 Statistiques des vues
- 🔔 Notifications de publication

### Fonctionnalités techniques
- 💾 Stockage local (LocalStorage)
- 📱 Design responsive "mobile-first"
- 🎨 Interface aux couleurs du Congo
- 🌐 Déploiement sur GitHub Pages
- ⚡ Pas de base de données nécessaire
